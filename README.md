# Сайт-резюме Frontend-разработчика

Персональное портфолио и сайт-резюме, созданное для демонстрации навыков, опыта и проектов.

## О проекте

Это одностраничное веб-приложение, разработанное с использованием современных технологий фронтенд-разработки. Сайт представляет собой интерактивную визитку, где можно узнать обо мне, моём опыте, используемых технологиях и связаться для сотрудничества.
